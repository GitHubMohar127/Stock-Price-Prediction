## Overview
An end-to-end machine learning project that predicts stock prices using real-time API data and displays results via a Streamlit frontend.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly

## Features
- Real-time data fetching via API
- Daily automated pipeline
- ML model training per stock
- Interactive frontend dashboard
